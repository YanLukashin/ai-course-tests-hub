# AI Course Tests Hub

Минимальный репозиторий для публикации межмодульных тестов курса ИИ на GitHub Pages.

Что входит в репозиторий:
- статическая платформа из папки `Платформа курса ИИ`;
- markdown-тесты модулей 1–6;
- GitHub Actions workflow для автоматической публикации на GitHub Pages.

Основная рабочая инструкция находится в:

- `Платформа курса ИИ/README.md`
